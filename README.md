# 📊 Amazon Sales Analysis

## 📌 Project Overview
This project analyzes Amazon sales data using Microsoft Excel and Python. It provides insights into sales performance, customer behavior, payment preferences, product categories, and regional sales through an interactive Excel dashboard and Python analysis.

## 🛠️ Tools Used
- Microsoft Excel
- Python
- Pandas
- Matplotlib
- Google Colab

## 📊 Key Business Questions
- Monthly Sales Trend
- Top 10 Products by Sales
- Top 5 Customers by Revenue
- Highest and Lowest Sales by State
- Category-wise Revenue Analysis
- Most Preferred Payment Method
- Average Order Value (AOV)
- Business Recommendations

## 📂 Files Included
- **amazon.xlsx** – Interactive Excel Dashboard with Pivot Tables, Charts, and Slicers
- **Mini_project.ipynb** – Python analysis performed in Google Colab

## 📈 Key Insights
- Identified the highest revenue-generating product category.
- Analyzed state-wise sales performance.
- Determined the most preferred payment method.
- Identified the top customers and products.
- Created an interactive dashboard for business decision-making.

## 🎯 Conclusion
This project demonstrates how Excel and Python can be combined to analyze sales data, generate business insights, and support data-driven decision-making.
